## Prerequisites

Before you begin, make sure you have:

- Node.js 18.0.0 or higher installed
- An API key (get one by [creating an account](/authentication) and generating an API key)
- Basic knowledge of HTTP requests and JSON

## Step 1: Get Your API Key

First, you need to authenticate and create an API key. See our [Authentication guide](/authentication) for detailed instructions.

For now, you'll need:
- A user account (create one via the login endpoint)
- An API key generated from your account

## Step 2: Make Your First Request

Let's start with a simple profile search. This endpoint searches for Facebook profiles matching your criteria.

<CodeGroup>

```bash cURL
curl -X POST https://api.proofly.joinplank.com/api/profiles/search \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-api-key-here" \
  -d '{
    "name": "John Doe",
    "location": "New York, NY",
    "birthYear": 1990
  }'
```

```javascript JavaScript
const response = await fetch('https://api.proofly.joinplank.com/api/profiles/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your-api-key-here'
  },
  body: JSON.stringify({
    name: 'John Doe',
    location: 'New York, NY',
    birthYear: 1990
  })
});

const data = await response.json();
console.log('Job ID:', data.jobId);
```

```typescript TypeScript
const response = await fetch('https://api.proofly.joinplank.com/api/profiles/search', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-Key': 'your-api-key-here'
  },
  body: JSON.stringify({
    name: 'John Doe',
    location: 'New York, NY',
    birthYear: 1990
  })
});

const data = await response.json() as { jobId: string };
console.log('Job ID:', data.jobId);
```

</CodeGroup>

## Step 3: Check Job Status

The API returns a job ID because profile searches are processed asynchronously. Poll the job status endpoint to get results:

<CodeGroup>

```bash cURL
curl -X GET https://api.proofly.joinplank.com/api/jobs/your-job-id-here \
  -H "X-API-Key: your-api-key-here"
```

```javascript JavaScript
const jobId = data.jobId;

// Poll for job status
const checkStatus = async () => {
  const response = await fetch(`https://api.proofly.joinplank.com/api/jobs/${jobId}`, {
    headers: {
      'X-API-Key': 'your-api-key-here'
    }
  });
  
  const status = await response.json();
  return status;
};

// Poll every 2 seconds until completed
const pollJob = async () => {
  while (true) {
    const status = await checkStatus();
    console.log('Status:', status.status);
    
    if (status.status === 'completed') {
      console.log('Results:', status.resultData);
      break;
    } else if (status.status === 'failed') {
      console.error('Job failed:', status.errorMessage);
      break;
    }
    
    await new Promise(resolve => setTimeout(resolve, 2000));
  }
};

pollJob();
```

```typescript TypeScript
interface JobStatus {
  id: string;
  status: 'queued' | 'processing' | 'completed' | 'failed' | 'waiting_for_browser';
  jobType: string;
  resultData?: any;
  errorMessage?: string;
}

const jobId = data.jobId;

// Poll for job status
const checkStatus = async (): Promise<JobStatus> => {
  const response = await fetch(`https://api.proofly.joinplank.com/api/jobs/${jobId}`, {
    headers: {
      'X-API-Key': 'your-api-key-here'
    }
  });
  
  return await response.json() as JobStatus;
};

// Poll every 2 seconds until completed
const pollJob = async () => {
  while (true) {
    const status = await checkStatus();
    console.log('Status:', status.status);
    
    if (status.status === 'completed') {
      console.log('Results:', status.resultData);
      break;
    } else if (status.status === 'failed') {
      console.error('Job failed:', status.errorMessage);
      break;
    }
    
    await new Promise(resolve => setTimeout(resolve, 2000));
  }
};

pollJob();
```

</CodeGroup>

## Step 4: Using the SDK (Recommended)

For a better developer experience, use our official SDK:

<CodeGroup>

```bash npm
npm install @plank-proofly/api
```

```bash yarn
yarn add @plank-proofly/api
```

```bash pnpm
pnpm add @plank-proofly/api
```

</CodeGroup>

```typescript
import { PlankProofly } from '@plank-proofly/api';

const client = new PlankProofly({ 
  apiKey: 'your-api-key-here' 
});

// Submit a profile search
// The SDK handles job polling automatically
const result = await client.profiles.search({
  name: 'John Doe',
  location: 'New York, NY',
  birthYear: 1990
});

console.log('Results:', result);
```

See our [SDK guide](/guides/sdk) for more examples and advanced usage.
